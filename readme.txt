== Description ==

Bizuno Locale, by PhreeSoft, provides language packs and locale tools for use within the Bizuno Accounting plugin. Note: We always welcome volunteer assistance with locale conversion. Please contact us if you are interested.

* Includes language translations for es_ES, fr_CA, nl_NL, and ro_RO to date, more may be available.
* Translator assistant to help make translating to your native language a snap.
* Contains additional fonts for reports and forms.

== Installation ==

Follow the standard installation procedure for all WordPress apps. Once activated the list of languages will appear in the login screen and can also be set in your profile.

= Minimum Requirements =

* Bizuno Accounting plugin
* PHP   version 5.4 or greater (PHP 5.6 or greater is recommended, tested with PHP 7.4)
* MySQL version 5.0 or greater (MySQL 5.6 or greater is recommended, tested with MySQL 5.6 & 5.7)
* WordPress 4.4+

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/bizuno-locale` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress.

= How can I translate Bizuno to my native language? =

* This plugin contains a tool that assists with translating to your native language. Currently there are about 2000 terms to translate, most contain a single or a few words.
